## Questions

1. Every `for` loop in Python secretly uses the iterator protocol (`iter()` and `next()`). Explain in your own words how a `for` loop actually works at the lower level. Why do you think Python hides this complexity from beginners by default? What advantage does understanding it give you as an intermediate developer?

2. The `datetime` module provides `date`, `time`, and `datetime` classes. Explain the difference between these three types, and describe a real application (such as a booking system, a payroll tool, or a subscription service) where you would need all three. What could go wrong if you used the wrong type for the wrong job?

3. `strftime` and `strptime` are mirror operations. Explain what each one does, why the names are confusing to many beginners, and describe a real situation — such as processing data from an external API or reading a CSV file — where you would need to use `strptime` to convert a string into a workable date object.

4. The `math` module provides `math.ceil()` and `math.floor()` alongside the built-in `round()`. Explain the difference between these three rounding methods and give a real business example — such as pricing, inventory, or billing — where each one is the correct choice and using the wrong one would produce an incorrect result.
