## Problem 2 — Find Customer Referee

- **Link:** https://leetcode.com/problems/find-customer-referee/
- **Difficulty:** 🟢 Easy
- **Why this lesson:** This problem teaches you that `NULL` in SQL is not the same as zero or empty string — a concept you will encounter constantly when working with PostgreSQL. Filtering NULL values correctly is a fundamental skill.
- **Hint:** A `WHERE` clause that checks `column != 2` will silently skip NULL rows. You need an extra condition to include them.
- **Your file:** `leetcode_2.sql`
