## Problem 1 — Reverse Linked List

- **Link:** https://leetcode.com/problems/reverse-linked-list/
- **Difficulty:** 🟢 Easy

## Problem 2 — Min Stack

- **Link:** https://leetcode.com/problems/min-stack/
- **Difficulty:** 🟢 Easy
- **Why this lesson:** This problem asks you to design a class with a private internal state and a clean public interface (`push`, `pop`, `top`, `getMin`) — the textbook definition of encapsulation.
- **Hint:** Think about how to track the minimum efficiently without scanning the whole stack each time. Consider maintaining a second private structure in parallel.
- **Your file:** `leetcode_2.py`
