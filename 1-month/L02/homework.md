## Questions

1. Python is a dynamically typed language, meaning you do not need to declare the type of a variable before using it. Some other languages (like Java or C++) require you to declare the type. Compare these two approaches — what are the benefits and risks of dynamic typing for a beginner? For a large professional project?

2. Why is it important to choose the correct data type when storing information? Describe a real-world situation — for example in a banking app, a school system, or an online store — where using the wrong data type (such as storing a number as a string) could cause a serious problem.

3. Variable naming matters more than most beginners realize. Compare these two versions of the same code and explain which is better and why:
   - Version A: `a = 25`, `b = 1.72`, `c = True`
   - Version B: `age = 25`, `height_meters = 1.72`, `is_verified = True`

4. The `bool` type only holds two values: `True` or `False`. Yet it is described as one of the most important data types in programming. Explain why booleans are so powerful. Give three real examples from an app or website you use where a True/False value is making a decision behind the scenes.
