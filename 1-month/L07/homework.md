## Questions

1. Explain the difference between a list and a tuple in Python. Beyond just saying "lists are mutable and tuples are immutable," give a specific real-world example of data that should be stored in a list and data that should be stored in a tuple — and explain *why* each choice is correct.

2. `sorted(lst)` and `lst.sort()` both sort a list, but they behave differently. Describe exactly what each one does, what it returns, and when you would choose one over the other. What bug could occur if a developer confused the two?

3. Negative indexing allows you to access list elements from the end. Describe why this feature is useful — give a real scenario where you would need to access the last item, the second-to-last item, or the last 3 items of a list without knowing the list's length in advance.

4. When you remove an item from a list, Python offers three methods: `.remove()`, `.pop()`, and `del`. Compare all three — what are the differences in how they identify the item to remove, what they return, and when each one is the right tool to use?
