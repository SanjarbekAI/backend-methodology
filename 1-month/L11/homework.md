## Questions

1. In today's project you combined variables, loops, conditions, lists, dictionaries, user input, and file I/O all in one program. Looking back at the last 10 lessons, which concept felt the most natural to use during the project and which one caused you the most difficulty? Explain why for each.

3. The project saves data to a file and loads it on the next run. Why is persistent storage important for a real application? Describe what the user experience would be like without it, and describe two other real applications where persistent storage is absolutely essential.

4. During the project, you had to validate user input in multiple places. Describe your validation strategy — what kinds of invalid input did you handle, and what approach did you use? If you were building this for 10,000 users instead of just yourself, what additional validation would you add?
