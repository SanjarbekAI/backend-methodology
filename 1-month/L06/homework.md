## Questions

1. Explain the difference between a `for` loop and a `while` loop. Describe a real-world process that is better represented by a `for` loop, and a separate real-world process that is better represented by a `while` loop. Explain your reasoning for each choice.

2. An infinite loop is one of the most common bugs beginners write. Describe exactly what an infinite loop is, what causes it, how you would recognize it if your program had one, and what you would do to fix it.

3. Compare `break` and `continue`. Many beginners mix them up. Explain precisely what each one does with a specific example for each. Then describe a real scenario — such as a search system, a checkout process, or a data filter — where you would use `break`, and a different scenario where you would use `continue`.

4. Nested loops multiply execution time. If an outer loop runs 100 times and the inner loop runs 100 times, how many total iterations happen? Describe a real situation where nested loops are necessary (not just a math problem), and explain what you would need to be careful about when using them.
