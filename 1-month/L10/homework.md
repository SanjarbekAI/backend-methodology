## Questions

1. Explain the difference between the file modes `"r"`, `"w"`, and `"a"`. Describe a real scenario for each mode — a situation where that specific mode is exactly the right choice and using the wrong one would cause a serious problem. For example, what would happen if you opened a log file in `"w"` mode instead of `"a"` mode?

2. The `with` statement is the recommended way to open files in Python. Explain what `with` actually does behind the scenes (in simple terms), and describe what could go wrong in a real program if you opened files manually with `open()` and `close()` without using `with`.

3. Python comes with a large standard library of modules. Explain what a module is, what problem it solves, and why it would be a bad idea to write your own square-root function, date calculator, or random number generator when Python's standard library already provides these. What is the risk of not knowing what's already available?

4. When you use `import math`, you access functions as `math.sqrt()`. When you use `from math import sqrt`, you access it as `sqrt()` directly. Compare these two approaches — what are the pros and cons of each? Is there a situation where one is clearly better than the other?
